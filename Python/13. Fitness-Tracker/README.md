# Fitness Tracker Project

This Python projects created using the **Requests module** concepts.  
This project allows users to input the physical exercise they performed during the day. The application estimates the calories burned and then automatically logs the data into Google Sheets using another API

---

## Projects Included
User input for daily physical exercise
Estimates calories burned using fitness API
Automatically logs data to Google Sheets
Stores exercise name, duration, calories, and date
Secure handling of API keys using environment variables
Clean and modular Python code

---

## Technologies Used
- Requests (API handling)
- dotenv / os (Environment variables)
- REST APIs & HTTP Headers
