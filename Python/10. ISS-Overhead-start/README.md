# ISS Overhead  Project

This Python projects created using the **Requests module**, **SMTP module** and **Object-Oriented Programming (OOP)** concepts.  
This project fetches data from a website using an API and sends the processed information via email using SMTP. Sensitive credentials are securely managed using environment variables

---

## Projects Included
- Fetches real-time data from an external API
- Processes and formats the fetched data
- Sends emails automatically using SMTP
- Uses environment variables to protect credentials
- Clean and modular Python code
- Object-Oriented Programming (OOP)
---

## Technologies Used
- Requests (API handling)
- smtplib (Email sending)
- email.message
- dotenv / os (Environment variables)
